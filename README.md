# I.M Chatbot EN

## Description

The **I.M Chatbot** project is a thematic chatbot based on I.M, a member of the K-pop group **Monsta X**. I.M, whose real name is **Im Changkyun**, is a South Korean rapper, singer, and songwriter, known for his unique style and captivating personality. This chatbot allows users to chat with I.M, explore different responses based on the message's intent, and interact with him in an aesthetic environment inspired by the Monsta X universe.

## Requirements

The goal of the project is to create a chat platform where users can:

1. **Homepage**:
   - A form that allows the user to enter their name to start the conversation with I.M.
  
2. **Chat Interface**:
   - A real-time chat where users can send messages and receive responses from the chatbot, inspired by I.M's universe.
   - A feature to **change the mode** (e.g., dark mode, light mode).
   - The ability to **change the bot's language** for different responses in various languages.
   - **Emojis** to enhance the chat experience.
   - An option to **log out** and end the session.

## Tools Used

The development of this chatbot is based on the following technologies and tools:

- **HTML, CSS, JavaScript**: for developing the interface and handling user interactions.
- **PHP & AJAX**: for managing asynchronous requests and retrieving data from the server.

For information related to IBM Watson Assistant certification and resources, you can refer to this link:  
[IBM Build Your Own Chatbot](https://www.ibm.com/cloud/learn/chatbots-explained).

# I.M Chatbot FR

## Description

Le projet **I.M Chatbot** est un chatbot thématique basé sur I.M, un membre du groupe de K-pop **Monsta X**. I.M, de son vrai nom **Im Changkyun**, est un rappeur, chanteur et compositeur sud-coréen, connu pour son style unique et sa personnalité captivante. Ce chatbot permet aux utilisateurs de discuter avec I.M, d'explorer différentes réponses en fonction de l'intention du message et d'interagir avec lui dans un environnement esthétique inspiré de l'univers de Monsta X.

## Cahier des charges

Le projet a pour objectif de créer une plateforme de chat où les utilisateurs peuvent :

1. **Page d'accueil** :
   - Un formulaire permettant à l'utilisateur d'entrer son nom pour démarrer la conversation avec I.M.
  
2. **Interface de chat** :
   - Un chat en temps réel où les utilisateurs peuvent envoyer des messages et recevoir des réponses du chatbot, inspirées de l'univers de I.M.
   - Une fonctionnalité permettant de **changer le mode** (exemple : mode sombre, mode clair).
   - La possibilité de **changer la langue** du bot pour différentes réponses dans diverses langues.
   - Des **émoticônes** à ajouter aux messages pour améliorer l'expérience de chat.
   - Une option pour **se déconnecter** et quitter la session.

## Outils utilisés

Le développement de ce chatbot se base sur les technologies et outils suivants :
